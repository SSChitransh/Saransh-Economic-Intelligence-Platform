<h2 style="color: #e8edf3; font-size: 2rem;">🔬 <span style="color: #f5c518;">Key Research Finding</span></h2>

<div style="background: #1a212b; border-left: 6px solid #f5c518; border-radius: 12px; padding: 20px 30px; max-width: 850px; margin: 0 auto; text-align: left; border: 1px solid #1e2733;">
  <p style="font-size: 1.1rem; color: #e8edf3; margin: 0;">
    <strong style="color: #f5c518;">🇮🇳 India</strong> shows the strongest negative correlation between education spending and poverty reduction 
    (<strong style="color: #f5c518;">r = -0.461</strong>, p < 0.001).
  </p>
  <p style="color: #8892a0; margin: 4px 0 0 0; font-size: 0.95rem;">

  </p>
  <p style="color: #b0b8c8; margin: 8px 0 0 0; font-size: 0.9rem;">
    💡 <strong>Conclusion:</strong> Education investment is a powerful tool for poverty alleviation in <strong style="color: #f5c518;">developing economies</strong>.
